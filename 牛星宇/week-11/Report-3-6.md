### 一、核心技术栈掌握情况
已掌握/落地的技术栈及核心能力可归纳为以下维度：

#### 1. Vue 核心基础（入门→工程化应用）
- **基础语法与核心思想**：掌握“数据驱动视图”核心逻辑，能完成 Vue 实例初始化（`el` 挂载、`data` 数据模型定义）、插值表达式 `{{ }}` 渲染（变量/算术/三元运算）、`v-model` 双向绑定（表单元素专属），解决过实例挂载失败、`v-model` 误用等入门级问题；
- **工程化开发**：熟悉 Vue 项目工程化架构，掌握路由配置（`src/router/index.js`，含路由懒加载、重定向、路由跳转）、根组件 `App.vue` 与 `<router-view>` 动态渲染、入口文件 `main.js` 配置（组件库全局注册）；
- **生命周期**：了解 `mounted` 钩子函数（用于异步请求拉取数据）。

#### 2. UI 组件库（Element UI）
- 掌握 Element UI 核心组件使用：`el-container`/`el-aside`/`el-menu`（布局/导航）、`el-table`（数据表格渲染，含自定义字段逻辑）、`el-form`（查询表单）、分页组件、对话框等；
- 能基于 Element UI 完成业务页面搭建（部门/员工管理页面），实现布局、样式优化及基础交互绑定。

#### 3. 前后端交互（Axios）
- **axios 全流程配置**：搭建全局 axios 实例，配置基准路径、超时时间、请求头；封装请求/响应拦截器（请求自动带 token、响应数据格式化、异常统一捕获，如 401/404/500 等）；
- **通用方法封装**：封装 get/post/put/delete 通用请求函数，适配 80% 以上接口调用场景，支持参数拼接、加载状态提示、自定义异常提示；
- **接口联调**：能调用远程接口获取数据并绑定至前端页面（如员工表格数据），解决过接口响应格式不统一、跨域等问题，维护接口文档。

#### 4. 业务开发能力
- 完成 tlias 宝可梦学习辅助系统核心模块（部门/员工管理）的前端开发：包括页面布局、路由配置、数据渲染、基础交互（查询按钮事件、分页事件绑定）；
- 能处理静态数据（JSON 文件）、模拟数据渲染，具备前端业务逻辑设计思路（如查询条件过滤、分页功能预留、编辑/删除按钮交互规划）。
